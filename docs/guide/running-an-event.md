# Running an Event

When you open an event, you enter the **event workspace** — the surface where you actually
run the race day. The workspace is a left-to-right sequence of **stage pages** you move
through and can revisit at any time. There is no separate "setup" mode: setting an event up
is just visiting its early stage pages.

A guided **setup wizard** is a quick first pass over the common path — it walks you through
the same early stage pages (picking classes, building the roster, choosing a format) so a
